    • Audio Compression
      Audio Compression
      Music and sound effects audio files are normalized to -3db in Audacity. There is an exception for audio that is supposed to be more quiet or soft, in those cases the files are normalized to -8db.

      The OGG file format is used for all audio, with the quality set to 5 of 10. I did listening tests and found this quality setting to be the lowest where I could not hear distortion.

    • Coding Reminders
      Coding Reminders
      “< = >”

      “Every comment is a failure to explain yourself in code.”

      Instead of making a comment to explain a complex conditional, make a well-named function.

      You are always forgetting vector::reserve, constexpr, and delegating constructors…stop that.

      Class names should be nouns, and should avoid: Manager, Processor, Data, Info, Object, Part, Entity.

      Prefer std::vector, but default to using std::array first. (It can be optimized away…)
